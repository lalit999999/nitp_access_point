CSE Academic Resource Platform
(Frontend – React + Tailwind + Fetch API)

Build a modern, highly responsive frontend web application for CSE students to browse and access semester-wise academic resources such as PYQs, Notes, and Books.

🛠 Tech Stack

React (Vite preferred)

Tailwind CSS

React Router

Context API (global state management)

Fetch API (for backend communication)

Environment variables for API base URL

The frontend must consume data from a separate backend REST API using Fetch.

🎨 UI & DESIGN REQUIREMENTS
🌗 Dark / Light Mode

Toggle switch in navbar

Store preference in localStorage

Use Tailwind dark: classes

Smooth transition between themes

Default to system theme on first visit

Apply transition-colors duration-300

2️⃣ Skeleton Loading (Animated)

Show skeleton while fetching data.

SkeletonCard Component
function SkeletonCard() {
  return (
    <div className="animate-pulse p-4 rounded-2xl shadow-md bg-gray-200 dark:bg-gray-800">
      <div className="h-40 bg-gray-300 dark:bg-gray-700 rounded-lg mb-4"></div>
      <div className="h-4 bg-gray-300 dark:bg-gray-700 rounded w-3/4 mb-2"></div>
      <div className="h-4 bg-gray-300 dark:bg-gray-700 rounded w-1/2"></div>
    </div>
  );
}

Usage
{loading ? (
  <div className="grid gap-6 grid-cols-1 sm:grid-cols-2 lg:grid-cols-4">
    {[...Array(8)].map((_, i) => (
      <SkeletonCard key={i} />
    ))}
  </div>
) : (
  <ResourceGrid />
)}

Best Practices:

Match skeleton count with grid layout

Use soft grays in light mode

Use darker shades in dark mode

Never show blank screen while loading

      </h2>

        Try adjusting your filters or search keyword.
      </p>
    </div>
  );
}

Usage
{resources.length === 0 && !loading && <EmptyState />}


Enhancements:

Add “Clear Filters” button

Keep large vertical spacing (py-20)

Use subtle fade-in animation

4️⃣ Sticky Search Bar

Keep search visible while scrolling.

Implementation
<div className="sticky top-0 z-40 
                bg-white dark:bg-gray-900 
                backdrop-blur-md 
                border-b 
                border-gray-200 dark:border-gray-700 
                shadow-sm">
  <SearchBar />
</div>

Why It Works:

sticky top-0 → sticks to top

z-40 → prevents overlap

backdrop-blur-md → modern glass effect

shadow-sm → premium appearance

UX Improvements:

Expand search input on focus

Debounce search input (300ms)

Show filter chips below search

Keep filter dropdown attached
